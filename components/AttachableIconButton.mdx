## AttachableIconButton

AttachableIconButton はカスタムボタンコンポーネントです。

### Props
- `type`でボタンの色を切り替えることができます。
- `callback`に関数を渡すことができます
- `leftIcon`にReactNodeを渡すと左側に表示されます（アイコン表示用途）
- `rightIcon`にReactNodeを渡すと右側に表示されます（アイコン表示用途）

### Main

```tsx
const callback = () => alert('callback');
<AttachableIconButton type="main" callback={callback}>Main</AttachableIconButton>
```
